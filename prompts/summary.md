エグゼクティブサマリーをJSON形式で作成してください。

## 出力JSON形式（厳密に従うこと）
{{
  "overview": "今週の動向を4-5文で要約。専門家の視点で、最も重要な動きから順に記述する。",
  "key_highlights": [
    {{
      "label": "ハイライトの短いタイトル（10文字以内）",
      "detail": "1文の具体的な説明",
      "article_ref": "#01"
    }}
  ],
  "categories": [
    {{
      "name": "カテゴリー名",
      "count": 3,
      "summary": "このカテゴリーの動向を1-2文で要約",
      "article_refs": ["#03", "#04", "#05"]
    }}
  ],
  "timeline": [
    {{
      "period": "2026前半",
      "event": "予定されるイベントやマイルストーン"
    }}
  ],
  "outlook": "今後2-3年の展望を3-4文で。具体的な数値予測や技術動向を含める。",
  "stats": {{
    "total_articles": 0,
    "countries": 0,
    "key_metrics": [
      {{"label": "指標名", "value": "値"}},
      {{"label": "指標名", "value": "値"}}
    ]
  }}
}}

## 重要ルール
- key_highlights は最大5件、最も注目度の高いものを選ぶ
- categories は記事の内容に基づき3-5カテゴリーに分類（例: OEM動向、量産化・商用化、技術開発、R&D基礎研究、市場分析）
- timeline は今後の主要マイルストーンを時系列で3-5件
- article_ref は記事番号（#01, #02...）で参照
- stats.key_metrics は業界の重要な定量データを2-4件
- すべて日本語で記述すること
