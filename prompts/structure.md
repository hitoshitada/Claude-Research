## 対話フォーマット

各台詞は以下の形式で書いてください：

```
F[感情]: 台詞テキスト
M[感情]: 台詞テキスト
```

### 感情タグ一覧
- `[N]` — ニュートラル（通常の語り）
- `[H]` — ハッピー/嬉しい/楽しい（明るいトーン）
- `[E]` — 興奮/驚き/感嘆（テンション高め）
- `[S]` — 真剣/シリアス/考え込む（落ち着いたトーン）
- `[Q]` — 質問/疑問（語尾上がり）
- `[T]` — ツッコミ/冗談（軽快なトーン）

### 感情タグのルール
- 全ての台詞に必ず感情タグを1つ付けること
- 同じ感情タグを3回以上連続で使わないこと（単調になるため）
- **スージー**: [E][Q][H][T]を多めに使い、感情豊かに表現する
- **トロイ**: [N]と[S]を中心に使うこと。[H]は控えめに。[E]は使わない。常に落ち着いたトーン
- 1行が1つの台詞です。改行で台詞を区切ってください
- 空行は入れないでください

### 例
```
F[H]: 皆さんこんにちは！テックトレンドへようこそ。ナビゲーターのスージーです。
M[N]: こんにちは。素材と製造の専門家、トロイです。今週もよろしくお願いします。
F[E]: トロイさん、今週もすごいニュースがありましたね！
M[N]: ええ。今週は特に、全固体電池の分野で注目すべき発表がありました。
F[Q]: え、どんな発表なんですか？気になります！
M[N]: フィンランドのスタートアップが、希少材料を一切使わずに超高速充電できる全固体電池を開発したんですよ。
F[E]: えー！それってすごいことじゃないですか！レアアース不要ってことですか？
M[S]: そうです。資源依存のリスクを大幅に下げながら、性能を確保した点が画期的なんですね。
```

※ トロイ（M）は常に落ち着いたトーン。[E]（興奮）は使わない。専門的な情報・数値・トピック紹介は必ずMが担当する。
